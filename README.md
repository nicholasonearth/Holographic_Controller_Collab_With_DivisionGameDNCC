# Computer Vision Hand Controller untuk Subway Surfers

Project ini adalah implementasi Computer Vision menggunakan Python dan MediaPipe untuk mengontrol game PC (khususnya Subway Surfers) menggunakan gestur tangan melalui webcam.

## Fitur Gestur
- **Lompat (Panah Atas):** Buka seluruh jari telapak tangan.
- **Turun/Rolling (Panah Bawah):** Kepalkan seluruh jari tangan.
- **Geser Kiri (Panah Kiri):** Buka jari telunjuk saja.
- **Geser Kanan (Panah Kanan):** Buka jari telunjuk dan jari tengah (Gestur "Peace" / V).

## Persyaratan Sistem
- Python 3.11 atau 3.12 (Disarankan).
- Webcam yang berfungsi.
